# hexo-tag-ltr

Force LTR layout direction when used in a mixed RTL and LTR layouts

# About
> This is a [Hexo](http://hexo.io/) tag plugin for changing the direction of your writing (layout)

## Installation
```bash
npm install --save hexo-tag-ltr
```

## Usage

```bash
{% ltr %}
  LTR text
{% endltr %} 
```
